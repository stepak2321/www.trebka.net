# trebka.net


## Features

### Core Functionality
- **Dynamic Status Display**: Real-time status updates with various system states
- **Theme Toggle**: Switch between light and dark modes (persisted in localStorage)
- **Interactive Canvas**: Animated particle system with connecting lines
- **Audio System**: Ambient background music and interactive sound effects
- **Terminal Interface**: Command-line interface with multiple commands

### Visual Effects
- **Matrix Rain**: Toggleable Matrix-style falling characters (Ctrl+Cmd+R)
- **Particle System**: Floating particles with physics (Ctrl+Cmd+P)
- **HUD Display**: System monitoring overlay with real-time stats
- **Responsive Design**: Mobile-friendly layout

### Interactive Elements
- **Secret Triggers**: Click the image 6 times to unlock admin mode
- **Konami Code**: Enter ↑↑↓↓←→←→BA for rainbow effect
- **Terminal Commands**: Full command-line interface with history and auto-completion

### Keyboard Shortcuts
- `Ctrl/Cmd + K`: Clear terminal
- `Ctrl/Cmd + M`: Toggle HUD
- `Ctrl/Cmd + R`: Toggle Matrix rain
- `Ctrl/Cmd + C`: Reset canvas
- `Ctrl/Cmd + P`: Toggle particles
- `Space`: Randomize status
- `Enter`: Process terminal command
- `↑/↓`: Navigate command history

### Terminal Commands
- `help`: Show available commands
- `status`: Display current system status
- `time`: Show current date and time
- `clear`: Clear terminal
- `matrix`: Toggle Matrix rain effect
- `particles`: Toggle particle system
- `hud`: Toggle HUD display
- `admin`: Activate admin mode
- `ping`: Network connectivity test
- `whoami`: Display user information
- `ls`: List files
- `cat <filename>`: Display file contents
- `echo <text>`: Echo text to terminal

### Admin Mode Features
- Unlocks secret overlay
- Extended ARG (Alternate Reality Game) sequence
- Additional system monitoring
- Enhanced terminal capabilities
